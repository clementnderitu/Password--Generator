# 🔐 Password Generator

A simple, modern password generator built with **HTML, CSS, and JavaScript**.  
It creates secure random passwords and includes a clean UI with light/dark mode support.

---

## ✨ Features

- 🔑 Generate strong random passwords
- 📋 Copy to clipboard with one click
- 🌗 Light & Dark mode toggle (with persistence)
- 🎨 Clean, modern UI design
- ⚡ Smooth animations and interactions

⚙️ How It Works

Generates random passwords using a mix of:

Uppercase letters

Lowercase letters

Numbers

Symbols

Uses Math.random() to select characters

Displays two generated passwords at once

Allows copying directly to clipboard

🌙 Dark Mode

Toggle between light and dark themes

Theme preference is saved using localStorage

Automatically loads your last selected theme

📋 Copy Feature

Click the clipboard icon to copy password

Provides instant visual feedback
